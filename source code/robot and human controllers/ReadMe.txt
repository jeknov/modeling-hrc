teleop.cpp - controller of a human subject's avatar. 

auto-nonemotional.cpp and 
auto-emotional.cpp - both are controllers of a simulated robot avatar. 
					 Either one or another should be used in a task, never use both.

auto-nonemotional.cpp is a controller of a robot avatar that doesn't react emotionally to anything.

auto-emotional.cpp is a controller of a robot avatar that react emotionally to certain triggers.
